Hello, JSP world!

JSP(Java Server Page)란, Java코드(<% %> 또는 <%=%> )가 들어가있는 HTML코드를 말하며 서블릿의 복잡함을 좀 더 간단하게 사용할 수 있다.
서블릿보다 쉽고 작성하기 빠른 것이 가장 큰 장점이며 디자인 부분 html과 로직부분java를 분리시킬 수 있다.
프로그래머가 직접 코딩한 서블릿보다 최적화된 서블릿으로 생성시켜주므로 효율적인 코드가 만들어지며 Javabean의 사용이 쉽다.
웹 애플리케이션 상에서 변수의 사용가능한 범위 설정이 쉽다.

JSP

스크립팅 원소(scripting element)는 JSP페이지에서 자바코드를 직접 기술할 수 있게 하는 기능이다. 선언문은 '<%! 자바코드 %>'로 표현하고 멤버변수와 메소드를 선언하기 위해 사용한다. 스크립트릿은 '<% %>'로 표현하고 JSP에서 자바코드를 사용하기 위해 사용한다. 표현식은 '<%= %>'로 표현하고 자바에서 작성한 값을 출력하기 위해 사용함. JSP에서 주석은 '<%-- --%>'로 표현한다.

EL

EL(Expression Language)은 <%= %>또는 out.print()와 마찬가지로 값을 출력하고자 할때 사용한다. 사용문법은 `${}`이며, EL안에서 간단한 연산도 가능하다. 데이터를 화면에 출력하는 것 뿐만 아니라 커스텀 액션, 태그에서의 조건절이나 표현절로 사용되기도 한다. 만일 없는 데이터에 대해 표현하게 된다면 null이 아니라 비어있는 값을 출력한다.

액션태그

액션태그는 표준 액션태그와 커스텀 액션태그로 나뉜다. 표준 액션 태그는 JSP에서 기본으로 제공하여 JSP페이지 내에서 바로 사용이 가능하고 태그 앞에 접두어로 jsp: 이 붙는다. 커스텀 액션 태그를 사용하기 위해선 별도의 라이브러리가 필요하며 접두어는 사용자가 지정한 대로 쓸 수 있다. core, format, functions등 로직에 필요한 태그들로 구성되어있다.
